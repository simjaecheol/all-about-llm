# Manus AI

Manus AI는 단순한 챗봇을 넘어 사용자의 복잡한 목표를 스스로 계획하고 실행하는 **범용 자율형 AI 에이전트(General Purpose AI Agent)**입니다. 2025년 말 Meta에 인수된 이후 기술력이 더욱 고도화되었습니다.

## 주요 기능

- **자율적 작업 수행 (Autonomous Execution)**: "마케팅 플랜 수립"과 같은 추상적인 목표를 주면, 스스로 하위 작업을 분산하고 시장 조사, 콘텐츠 작성, 이미지 생성 등을 독립적으로 완료합니다.
- **My Computer (로컬 통합)**: 사용자의 PC(Windows/macOS) 내 로컬 파일에 접근하고 Excel, Photoshop 등 설치된 앱을 직접 제어하여 실무를 수행합니다.
- **실시간 리플레이**: AI가 가상 브라우저나 OS 내에서 작업하는 과정을 사용자가 실시간으로 모니터링하고 필요한 순간에 직접 개입할 수 있습니다.
- **멀티모달 결과물**: 텍스트 답변뿐만 아니라 실제 동작하는 웹사이트, 프레젠테이션 슬라이드, 인포그래픽 등을 직접 제작하여 결과물로 제공합니다.

## 기술적 특징

Manus AI는 클라우드의 강력한 지능과 사용자의 로컬 컴퓨팅 파워를 결합한 하이브리드 방식을 사용합니다. 이를 통해 외부에서도 스마트폰을 통해 집이나 사무실의 PC에 복잡한 연산이나 모델 학습 작업을 명령할 수 있는 유연성을 제공합니다.

### OpenManus와의 관계

**Manus AI**가 최신 기술과 고성능 인프라를 바탕으로 한 완성된 **상용 서비스**라면, **[OpenManus](./open_manus.md)**는 그 핵심 메커니즘을 오픈소스로 구현하여 사용자가 직접 제어하고 커스터마이징할 수 있게 하는 **오픈소스 커뮤니티 프로젝트**입니다. 사용자는 고도화된 성능이 필요할 때는 Manus AI를, 자유로운 커스터마이징과 개인화가 중요할 때는 OpenManus를 선택할 수 있습니다.
