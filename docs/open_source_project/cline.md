# Cline (구 Claude Dev)

Cline은 VS Code 확장 프로그램으로, AI가 단순한 코드 제안을 넘어 파일 생성, 터미널 명령 실행, 웹 브라우징까지 직접 수행하는 **에이전트형 코딩 도구**입니다.

## 주요 특징

- **에이전틱 코딩(Agentic Coding)**: "로그인 페이지를 만들어 줘"와 같은 명령을 내리면, 필요한 파일을 생성하고 패키지를 설치하며 실제 실행 결과까지 확인합니다.
- **BYOK (Bring Your Own Key)**: 사용자가 원하는 AI 모델 API(Claude 3.5 Sonnet 등)를 직접 연결하여 비용과 성능을 조절할 수 있습니다.
- **브라우저 통합**: AI가 문서를 직접 읽거나 웹 페이지를 확인하여 최신 API 정보를 기반으로 코드를 작성합니다.
- **VS Code와 긴밀한 통합**: IDE 내에서 직접적으로 에디터와 터미널을 조작하므로 개발 환경을 벗어날 필요가 없습니다.

## 특징 비교 (vs. Cursor)

| 구분 | Cursor | Cline |
| :--- | :--- | :--- |
| **형태** | AI 특화 독립 IDE | **VS Code 확장 프로그램** |
| **자율성** | 고도화된 코드 편집 | **파일/터미널 직접 제어 에이전트** |
| **비용 모델** | 월간 구독 중심 | API 사용량 기반 (BYOK) |

Cline은 개발자가 에이전트와 대화하며 마치 프로젝트 전체를 지휘하는 듯한 경험을 제공합니다.
