# 오픈소스 프로젝트

2025년 하반기 이후, 모델 자체보다는 그 모델을 활용하여 실질적인 문제를 해결하는 **AI 애플리케이션(AI Applications)** 중심의 오픈소스 프로젝트들이 큰 주목을 받고 있습니다. 이 섹션에서는 사용자가 직접 설치하고 활용할 수 있는 도구들을 중심으로 정리합니다.

## 주요 카테고리

### 자율형 에이전트 및 개인 비서 (Autonomous Agents)
- **[OpenManus](./open_manus.md)**: Manus AI의 오픈소스 구현체로, 사용자의 목표를 스스로 계획하고 실행하는 범용 에이전트입니다.
- **[OpenClaw](./open_claw.md)**: 메시징 앱과 연동되어 실제 업무(웹 서핑, 예약, 코드 실행 등)를 대행하는 개인용 AI 비서입니다.
- **[NanoClaw](./nanoclaw.md)**: 보안과 격리를 최우선으로 하는 OpenClaw의 보안 특화 대안 프로젝트입니다.

### AI 코딩 어시스턴트 (AI Coding Assistants)
- **[Aider](./aider.md)**: 터미널 기반의 AI 페어 프로그래머로, 기존 코드베이스를 이해하고 Git 커밋까지 자동으로 수행합니다.
- **[Cline](./cline.md)**: VS Code 내에서 파일 생성, 터미널 실행, 브라우징을 직접 수행하는 에이전트형 코딩 도구입니다.

### 지능형 검색 및 정보 수집 (Intelligent Search)
- **[Perplexica](./perplexica.md)**: 개인정보 보호를 강화한 AI 기반 오픈소스 검색 엔진으로, Perplexity의 대안으로 인기가 높습니다.
- **[Open WebUI](./open_webui.md)**: Ollama 등 로컬 모델을 위한 UI로, 최근 웹 검색 기능을 통합하여 프라이빗 검색 엔진 역할을 수행합니다.
