# OpenManus

OpenManus는 2026년 초 화제가 된 **Manus AI**의 기능을 오픈소스로 재구현하려는 프로젝트입니다. "누구나 자신만의 자율형 에이전트를 가질 수 있어야 한다"는 철학 아래 빠르게 성장하고 있습니다.

## 주요 특징

- **범용 작업 수행**: 사용자가 입력한 자연어 목표를 분석하여 브라우징, 데이터 분석, 문서 작성 등 다양한 작업을 자율적으로 수행합니다.
- **오픈소스 자유도**: 상용 서비스인 Manus AI와 달리, 사용자가 직접 모델을 선택(OpenAI, Claude, Ollama 등)하고 코드를 수정하여 자신만의 워크플로우를 구축할 수 있습니다.
- **로컬 및 클라우드 지원**: 개인 PC에서 실행하거나 클라우드 서버에 배포하여 사용할 수 있는 유연한 실행 환경을 제공합니다.
- **커뮤니티 중심 개발**: 전 세계 개발자들이 참여하여 새로운 도구(Tools)와 스킬을 지속적으로 추가하고 있습니다.

## Manus AI와의 관계

Manus AI가 고성능의 완성된 서비스를 제공한다면, OpenManus는 그 핵심 메커니즘을 투명하게 공개하고 사용자가 직접 제어할 수 있게 함으로써 오픈소스 에이전트 생태계를 확장하고 있습니다.
