# Open WebUI

<img src="../../assets/images/open_webui_logo.png" alt="Open WebUI Logo" width="150" />

Open WebUI(구 Ollama WebUI)는 Ollama 등 다양한 로컬 모델을 가장 쉽고 강력하게 활용할 수 있는 웹 기반 사용자 인터페이스(UI)입니다.

## 주요 기능

- **통합 검색 (Web Search)**: 모델이 최신 정보를 검색하여 답변할 수 있는 기능을 내장하고 있어, 자신만의 **프라이빗 Perplexity**를 구축할 수 있습니다.
- **다양한 모델 지원**: Ollama를 통한 로컬 모델은 물론, OpenAI, Claude API 등을 하나의 인터페이스에서 통합 관리합니다.
- **문서 분석 (RAG)**: PDF, 텍스트 파일 등을 업로드하여 AI와 함께 문서를 분석하고 질의응답을 수행할 수 있습니다.
- **사용자 관리**: 다중 사용자 지원 및 역할 기반 권한 제어가 가능하여 소규모 팀이나 조직에서도 활용하기 적합합니다.

## 기술적 장점

- **프라이버시 최우선**: 모든 대화 내용과 업로드한 문서를 로컬 서버나 사설 클라우드에만 저장하여 외부 데이터 유출을 방지합니다.
- **확장성**: 사용자 정의 함수(Functions)와 필터(Filters)를 통해 UI를 커스터마이징하거나 특정 기능을 추가할 수 있습니다.
- **Docker 기반 배포**: 복잡한 설정 없이 Docker 명령 한 줄로 간편하게 설치하고 운영할 수 있습니다.

Open WebUI는 단순한 챗봇 인터페이스를 넘어, 개인이 모든 데이터를 제어하며 사용하는 지능형 지식 센터로 진화하고 있습니다.
