# Dify (디파이)

Dify는 LLM 애플리케이션 개발을 위한 오픈소스 LLMOps 및 Agentic Workflow 플랫폼입니다. 시각적인 인터페이스를 통해 복잡한 AI 에이전트 워크플로우를 설계할 수 있는 것이 가장 큰 특징입니다.

## 주요 특징

- **시각적 워크플로우 빌더**: 드래그 앤 드롭 방식으로 복잡한 AI 파이프라인과 에이전트 워크플로우를 직관적으로 설계할 수 있습니다.
- **Agentic RAG**: 단순한 정보 검색을 넘어, 에이전트가 스스로 판단하여 적절한 지식을 검색하고 도구를 선택하는 고도화된 RAG 시스템을 지원합니다.
- **다양한 모델 지원**: OpenAI, Anthropic, Google, 오픈소스 모델(Llama 등)을 자유롭게 선택하여 통합할 수 있습니다.
- **운영 및 모니터링**: 에이전트의 실행 로그, 비용 분석, 응답 품질 평가 등 기업 수준의 운영 기능을 제공합니다.

## 2026년 업데이트 현황

2026년 3월 기준, Dify는 GitHub Star 약 13만 개를 달성하며 엔터프라이즈 AI 플랫폼 분야에서 선도적인 위치를 점하고 있습니다. 3천만 달러 규모의 Series Pre-A 투자를 유치하며 기업용 에이전트 생태계 확장에 박차를 가하고 있습니다.

## 활용 사례

- **고객 서비스 에이전트**: 복잡한 비즈니스 로직과 지식 베이스를 결합한 자율형 상담원 구축
- **데이터 분석 에이전트**: 데이터베이스와 직접 연결되어 사용자의 질문에 답하고 시각화 자료를 생성하는 에이전트
- **내부 도구 자동화**: 사내 ERP, Slack 등과 연동되어 반복적인 업무를 처리하는 워크플로우
