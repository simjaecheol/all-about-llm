# OpenFang

OpenFang은 단순한 에이전트 프레임워크를 넘어 **'에이전트 전용 운영체제(Agent OS)'**를 표방하는 혁신적인 프로젝트입니다.

## 주요 특징

- **7 Hands (7개 핵심 모듈)**: 에이전트의 지능을 스케줄링, 지식 그래프 관리, 도구 사용 등 7개의 독립적인 모듈('Hands')로 나누어 관리합니다.
- **분산 지식 저장소**: 에이전트가 학습한 정보를 중앙 집중식 DB가 아닌, 전용 노드 네트워크를 통해 안전하고 분산된 형태로 저장합니다.
- **모듈형 확장성**: 사용자가 필요한 'Hands' 모듈만 선택하여 자신만의 에이전트 커널을 구성할 수 있습니다.
- **자치적 운영**: 에이전트 간의 상호작용과 데이터 교환이 사전에 정의된 규칙에 따라 자율적으로 이루어지도록 설계되었습니다.

## 비전

OpenFang의 궁극적인 목표는 개인이나 기업이 자신만의 독립적인 'AI 에이전트 커널'을 소유하고, 이를 통해 안전하고 자율적인 디지털 자산을 관리하는 환경을 구축하는 것입니다.
