# OpenHands (구 OpenDevin)

<img src="../../assets/images/openhands_logo.png" alt="OpenHands Logo" width="150" />

OpenHands는 기존의 OpenDevin 프로젝트가 2024년 하반기 리브랜딩을 거쳐 진화한 것으로, 소프트웨어 엔지니어링 작업에 특화된 자율형 AI 에이전트 플랫폼입니다.

## 프로젝트의 목적

OpenHands는 사람이 직접 코드를 작성하는 대신, AI가 전체 개발 과정을 주도하거나 사람과 긴밀하게 협업하는 환경을 제공하는 것을 목표로 합니다. 단순한 자동 완성을 넘어, 계획 수립부터 구현, 테스트, 배포까지 전 과정을 자율적으로 수행할 수 있습니다.

## 주요 특징

- **소프트웨어 엔지니어링 특화**: 복잡한 코드베이스를 이해하고, 버그를 수정하며, 새로운 기능을 추가하는 데 최적화되어 있습니다.
- **자가 진화(Self-Evolution) 능력**: 프로젝트 코드의 약 20%가 OpenHands 에이전트 자신에 의해 개선되거나 수정되었을 정도로 고도화된 자율성을 보여줍니다.
- **샌드박스 실행**: 모든 코드 실행은 격리된 샌드박스 환경에서 이루어져 호스트 시스템의 안전을 보장합니다.
- **범용 에이전트 플랫폼**: 특정 모델에 종속되지 않고 다양한 LLM을 선택하여 활용할 수 있는 유연한 아키텍처를 갖추고 있습니다.

## OpenDevin에서 OpenHands로의 리브랜딩

2024년 8월, 프로젝트 명칭을 OpenHands로 변경하며 단순한 코딩 에이전트를 넘어 인간과 AI의 '협업'을 강조하는 방향으로 확장되었습니다. 이는 AI가 도구가 아닌 팀원으로서의 역할을 수행한다는 비전을 담고 있습니다.
