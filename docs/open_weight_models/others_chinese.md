# 기타 주요 중국 모델 (InternLM, MiniCPM, Baichuan)

중국의 오픈 웨이트 모델 생태계는 거대 테크 기업뿐만 아니라 학계와 스타트업을 중심으로 매우 다양하게 발전하고 있습니다.

## 1. InternLM (상하이 AI 랩)
*   **개요**: 상하이 AI 연구소(Shanghai AI Lab)에서 주도하는 오픈 소스 프로젝트입니다.
*   **주요 특징**:
    *   **InternLM-XComposer**: 7B 규모로 GPT-4V에 필적하는 이미지/비디오 분석 성능을 갖춘 멀티모달 특화 모델.
    *   **창의적 에이전트**: 웹페이지 제작, 시각적 스토리텔링 등 창의적인 작업에 최적화.
    *   **학술적 완성도**: 모델 아키텍처와 학습 데이터셋을 투명하게 공개하여 학술 연구에 널리 활용됨.

## 2. MiniCPM (OpenBMB / ModelBest)
*   **개요**: 칭화대학교의 OpenBMB 오픈 소스 커뮤니티에서 개발한 온디바이스 최적화 모델 시리즈입니다.
*   **주요 특징**:
    *   **MiniCPM-V 2.6**: 모바일 기기(iPhone, Android)에서 실시간 비디오 이해 및 추론이 가능할 정도로 가볍고 강력함.
    *   **효율성 극대화**: 적은 파라미터(2B~4B)로도 거대 모델에 가까운 성능을 내는 '가성비' 모델의 정점.
    *   **엣지 컴퓨팅**: 개인용 기기에서 클라우드 연결 없이 프라이버시를 보호하며 AI 기능을 구동하는 데 최적화.

## 3. Baichuan (바이촨)
*   **개요**: 중국 최대 검색 엔진 소고(Sogou)의 설립자 왕샤오촨이 설립한 Baichuan Intelligence에서 개발한 모델입니다.
*   **주요 특징**:
    *   **전문 도메인 강화**: 의료, 법률, 금융 등 전문 지식이 요구되는 분야에서 중국어 특화 성능이 매우 뛰어남.
    *   **Baichuan 3**: 수천억 파라미터급 대규모 모델로 진화하며 복잡한 논리 추론 능력 확보.
    *   **검색 엔진 기반 데이터**: 방대한 검색 데이터를 활용하여 최신 정보 업데이트와 사실 관계 정확도가 높음.

## 요약 및 비교

| 모델명 | 주요 강점 | 추천 시나리오 |
| :--- | :--- | :--- |
| **InternLM** | 멀티모달, 시각적 분석 | 영상 분석, 비전 에이전트 |
| **MiniCPM** | 온디바이스, 경량화 | 모바일 앱, 로컬 프라이빗 AI |
| **Baichuan** | 전문 지식, 중국어 최적화 | 의료/법률 분석, 중국어 특화 서비스 |
