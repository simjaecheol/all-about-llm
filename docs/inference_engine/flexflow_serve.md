# FlexFlow Serve

## 특징

### 낮은 지연시간, 고성능 LLM 서빙
- SpecInfer를 통한 추측 추론 및 토큰 트리 검증
- 단일 노드에서 1.3-2.0배, 멀티 노드에서 1.4-2.4배 성능 향상
- 분산 추론 최적화

## 기술적 접근

### 추측 추론 메커니즘
- 작은 추측 모델들을 조합하여 LLM 출력 예측
- 토큰 트리 기반 병렬 디코딩 메커니즘
- CPU 오프로딩 및 int4/int8 양자화 지원

### 분산 처리 최적화
- 멀티 노드 환경에서의 효율적인 작업 분산
- 네트워크 오버헤드 최소화
- 동적 로드 밸런싱

## 장단점

### 장점
- 멀티 노드 분산 추론 최적화
- 높은 처리량과 낮은 지연시간
- 효율적인 리소스 활용
- 확장 가능한 아키텍처

### 단점
- 복잡한 분산 설정
- 네트워크 인프라 요구사항
- 초기 설정 복잡성

## 사용 시나리오
- 멀티 노드 분산 추론 환경
- 대규모 LLM 서빙 인프라
- 높은 처리량 요구사항
- 분산 컴퓨팅 환경에서의 최적화
