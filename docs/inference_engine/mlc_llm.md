# MLC LLM

## 특징

### Apache TVM 기반 컴파일 최적화
- 브라우저, 모바일, CPU, GPU 등 다양한 플랫폼 지원
- WebLLM을 통한 브라우저 내 추론
- 하드웨어별 최적화

## 핵심 기술

### 크로스 플랫폼 지원
- 다양한 하드웨어 환경에서의 최적화
- 효율적인 리소스 사용
- 빠른 추론 속도
- 엣지 디바이스 최적화

### WebLLM 통합
- 브라우저 환경에서의 LLM 추론
- 클라이언트 사이드 AI 처리
- 네트워크 의존성 최소화

## 장단점

### 장점
- 하드웨어별 최적화
- 효율적인 리소스 사용
- 빠른 추론 속도
- 엣지 디바이스 최적화
- 크로스 플랫폼 지원

### 단점
- 특정 하드웨어 최적화 부족
- 복잡한 설정
- 제한적인 모델 지원

## 사용 시나리오
- 모바일 및 엣지 디바이스
- 브라우저 기반 AI 애플리케이션
- 다양한 하드웨어 환경
- 리소스 제약 환경
