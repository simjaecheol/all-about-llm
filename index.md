# All About LLM

LLM 서비스를 구축하기 위해 알아두면 좋은 지식들을 모아두는 공간입니다.

이 문서는 LLM(대형 언어 모델) 관련 핵심 개념, 실전 적용 방법, 최신 오픈소스 프로젝트 등을 정리하여, 입문자와 실무자 모두에게 도움이 되도록 구성되었습니다.
